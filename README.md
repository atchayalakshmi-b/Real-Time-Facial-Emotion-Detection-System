# Real-Time Facial Emotion Detection

This project detects human facial emotions in real time using a webcam.
It uses computer vision and deep learning techniques to recognize emotions
such as happy, sad, angry, surprised, and neutral.

## Features
- Real-time emotion detection using webcam
- Face detection with OpenCV
- Emotion classification using Deep Learning
- Simple and interactive interface

## Technologies Used
- Python
- OpenCV
- Deep Learning (CNN)
- Flask (for web integration, if applicable)

## How to Run
1. Clone the repository
2. Install required dependencies
3. Run the main Python file
4. Allow camera access

## Use Case
Useful for AI research, emotion analysis, human-computer interaction,
and smart surveillance systems.
